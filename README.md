# 👓 Lumière Landing Page

A modern and responsive landing page for a fictional optical clinic, built with semantic HTML and modern CSS.

<img width="1904" height="907" alt="Lumière-Landing-Page" src="https://github.com/user-attachments/assets/03b0db54-2b5c-4a1e-ac9d-9ba83dfb01ec" />

## 📖 About the project

Lumière is a fictional (for now) optical clinic focused on providing modern, accessible, and personalized eye care.
This is the landing page for a future project on the Lumière website, a personal project currently under development.

This project was created as part of my web development learning journey through The Odin Project. Its main goal was to strengthen my understanding of semantic HTML, modern CSS, responsive layouts, and reusable design systems.

---

## ✨ Features

- Responsive design
- Semantic HTML5
- CSS Variables (Design System)
- Flexbox
- CSS Grid
- SVG icons
- Hover animations
- Reusable button components
- Google Fonts integration
